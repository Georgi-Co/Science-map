# Science-map
Дипломный проект: Карта НИР

media.css - файл стилей медиазапросов для конфигурации адаптивности

author.html - страница подробной информации об авторе проекта
full-article.html - страница подробной информации выбранного проекта

script.js - основной скрипт проекта
author.js - скрипт логики страницы автора
full-article.js - скрипт логики страницы проекта
search.js - скрипт логики работы поисковой панели (поле ввода, фильтры, кнопки поиска и очистки фильтра)
pagination.js - скрипт логики работы пагинации списка проектов
accessibility.js - скрипт логики работы панели доступности (панель для слабовидящих, шрифт Брайля, цветовые схемы, звуковая навигация)
